# 跨会议小领域综合

本目录汇总 26 个 venue 的 second-wave subfield deep dive、paper evidence matrix、实体消歧、影响力审计和 workload/artifact inventory。主分类按具体研究问题和瓶颈组织，不按 FPGA、EDA、HPC、IC、ML systems 等大领域词组织。

## 产物

- `synthesis.md`：主报告。
- `cross_venue_subfield_inventory.csv`：12 个跨 venue 小领域。
- `representative_paper_matrix.csv`：代表论文矩阵，区分 origin / formation / milestone / tool-benchmark / frontier。
- `trend_claims.csv`：趋势判断及其 evidence IDs / paper IDs。

## 检查项

- 每条趋势判断必须能回到 venue evidence、paper IDs 或 workload/artifact inventory。
- `TOPIC_TAXONOMY.md` 只做状态级修订；除 T11 作为评价/证据基础设施标为 `stable` 外，其他 topic 保持 `provisional`、`split_candidate` 或 `merge_candidate`。
